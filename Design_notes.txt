CPU stages:
IF1 -> IF2 -> Decode -> Execute -> Memory -> WriteBack

IF1: Request pc <- current pc
IF2: Returns pc 

Following instructions are supported: 
    R-type: ADD, SUB, AND, OR, XOR, SLT
    I-type: ADDI, ANDI, ORI, SLTI, LW, JALR
    S-type: SW
    B-type: BEQ, BNE
    U-type: LUI
    J-type: JAL
